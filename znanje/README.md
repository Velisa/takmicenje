# Задаци
## Петље
### Наредба `break`
#### Задатак 1
Шта ће исписати сљедећи програм?
````
#include <iostream>
using namespace std; 
int main() {
    for (int i=0; i < 2; ++i) {
        for (int j=0; j < 5; ++j) {
            if (j == 2) 
                break;
            cout << i << j << " ";
        }
    }
    return 0;    
}
````
### Наредба `continue`
#### Задатак 1
Шта ће исписати сљедећи програм?
````
#include <iostream>
using namespace std; 
int main() {
    for (int i=0; i < 10; ++i) {
        if (i != 5)
            continue;
        cout << i << " ";
    }
    return 0;
}
````

## Кориштење функција
#### Задатак 1
Шта ће исписати сљедећи програм?
````
#include <iostream>
#include <cmath>
using namespace std; 
int main() {
  cout  << "round(+2.3) = " << round(2.3) << endl
        << "round(+2.5) = " << round(2.5) << endl
        << "round(+2.7) = " << round(2.7)  << endl
        << "round(-2.3) = " << round(-2.3) << endl
        << "round(-2.5) = " << round(-2.5) << endl
        << "round(-2.7) = " << round(-2.7) << endl
        << "round(-0.0) = " << round(-0.0) << endl
        << "round(-Inf) = " << round(-INFINITY) << endl;
  return 0;
}
````
