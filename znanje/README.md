# Задаци
## Петље
### Наредба `break`
#### Задатак 1
Шта ће исписати сљедећи програм?
````
#include <iostream>
using namespace std; 
int main() {
    for (int i=0; i < 2; ++i) {
        for (int j=0; j < 5; ++j) {
            if (j == 2) 
                break;
            cout << i << j << " ";
        }
    }
    return 0;    
}
````
### Наредба `continue`
#### Задатак 1
Шта ће исписати сљедећи програм?
````
#include <iostream>
using namespace std; 
int main() {
    for (int i=0; i < 10; ++i) {
        if (i != 5)
            continue;
        cout << i << " ";
    }
    return 0;
}
````
#### Задатак 2
Шта ће исписати сљедећи програм?
````
#include <iostream>

        for (int j=0; j < 5; ++j) {

        }
````
