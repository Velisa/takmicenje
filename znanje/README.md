* Логички оператори

### Вајл петља
```
while (uslov) {
  iskaz(i);
}
```
Да би се изрази унутар петље извршиле, услов мора бити тачан.
### Ду-вајл петља
```
do {
  iskaz(i);
} while (uslov);
```
Тијело петље се извршава најмање једном. Да би се петља извршила више него једном, услов мора бити тачан.
